# Benchmark Runner

Erzeugt ein KPI-Snapshot für den aktuellen Repo-Zustand.

## CLI

```bash
python src/benchmark_runner.py --output docs/benchmark_report.json
```

## KPIs

- total_skeletons
- avg_fitness
- top10_avg_fitness
- resonance_event_count
- resonance_coverage_ratio
- dominant_type_diversity
- readme/lifecycle integrity fields

# Repo Resonance Standard (IrsanAI)

Ziel: Für jedes Repo schnell sichtbar machen, wie gut einzelne Roadmap-Aspekte
mit dem Gesamt-Intent und der bestehenden Codebasis harmonieren.

## Kernmetriken

1. **Resonanz (0–100)**  
   Wie stark zahlt ein Task auf das Kernziel des Repos ein?

2. **Chemie / Repo-Fit (0–100)**  
   Wie gut passt der Task zu bestehenden Modulen, Datenflüssen und UX?

3. **Reifegrad (0–100)**  
   Wie stabil/testbar/operational ist die Umsetzung?

## Visualisierung

- Nutze pro Task:
  - numerischen Score (z. B. `83/100`)
  - 5er-Balken (`🟩🟩🟩🟩⬜`)
  - 1 Zeile Coach-Feedback

## Standard-Interpretation

- **85–100:** Elite / sehr stark integriert
- **70–84:** gut bis sehr gut
- **50–69:** mittel, mit klarem Ausbaupotenzial
- **0–49:** early stage / experimentell

## Mindestregel

Jede Roadmap in IrsanAI-Repos sollte unter jedem Punkt mindestens enthalten:
- Resonanz-Score
- Chemie-Score
- kurzes Coach-Feedback
